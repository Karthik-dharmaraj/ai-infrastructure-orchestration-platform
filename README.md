## AI-powered Kubernetes Infrastructure Orchestration Platform built using Kubernetes, FastAPI, React, VMware, Ubuntu, and Ansible.

## Features

- One-click Kubernetes deployment
- Infrastructure provisioning automation
- AI-driven deployment planning
- Kubernetes cluster management
- Health monitoring
- Scale in / scale out workflows
- Workflow orchestration
- Future AI Agent integration
- OpenStack integration roadmap
- Multi-cluster orchestration design
